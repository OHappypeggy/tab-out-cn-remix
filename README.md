# Tab Out 中文增强版

一个基于 Tab Out 的中文二创 Chrome 新标签页插件。

本项目基于 [@zarazhangrui](https://github.com/zarazhangrui) 的原项目 [tab-out](https://github.com/zarazhangrui/tab-out) 进行二创和个人功能增强，仅用于中文化与体验优化，不冒充原作者官方版本。原项目协议为 MIT，本仓库保留原作者署名与参考链接。

---

## 这个版本做了什么

- 将首页核心文案改为中文，包括问候语、模块标题、提示语和按钮文案
- 新标签页背景替换为更明显的毛玻璃视觉，并支持用户自定义上传壁纸
- 增加“我的书签”模块，支持展示 Chrome 书签栏和嵌套文件夹
- 支持书签和文件夹折叠展开
- 支持书签和文件夹重命名
- 支持书签和文件夹拖拽换位置
- 搜索框支持最近 5 条搜索历史下拉
- 增加“今日陪伴”提醒，会根据当天活跃浏览时长生成轻量提示文案
- 保留原版的标签页分组、重复标签检测、稍后处理、关闭动画和音效体验

---

## 功能概览

- 按站点分组展示当前打开的标签页
- 识别常见首页标签并单独归类
- 一键关闭单个标签页、某一组标签页或全部标签页
- 将页面加入“稍后处理”列表
- 查看和整理书签栏内容
- 直接在新标签页里重命名书签和文件夹
- 直接拖拽调整书签和文件夹顺序
- 点击搜索框时显示最近搜索记录
- 支持用户自己上传 JPG / PNG / WebP 作为新标签页背景
- 默认提供绿色护眼风格壁纸，上传后的壁纸设置保存在浏览器本地
- 支持根据今天的活跃浏览时长显示陪伴式提醒文案

## 安装方式

### 1. 克隆仓库

```bash
git clone <你的仓库地址>
```

### 2. 在 Chrome 中加载插件

1. 打开 `chrome://extensions`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择仓库里的 `extension/` 目录

### 3. 插件加载路径

Chrome 需要选择的目录是：

```text
extension/
```

也就是这个仓库根目录下的扩展目录，而不是仓库根目录本身。

---

## 自定义壁纸怎么用

加载扩展后，打开一个新标签页，就能在右上角看到一个低调的图片图标入口：

- 点击右上角图片图标，可以上传自己的 JPG、PNG 或 WebP 图片
- 图片会自动压缩成适合扩展存储的版本，并保存在当前浏览器本地
- 仓库默认提供的是一张绿色、偏护眼风格的壁纸

这意味着其他用户下载或 fork 这个项目后，不需要改源码，也能直接在插件界面里换自己的背景图。

---

## 今日陪伴提醒怎么统计

- 右上角壁纸图标左边有一个轻量小模块，会显示“今天已努力了多久”以及对应的提醒文案
- 统计口径不是读取完整浏览历史，而是记录 Chrome 窗口处于焦点中且用户保持活跃时的使用时长
- 这项统计会从安装或更新到当前版本后开始累计，不会回溯更早以前的浏览器使用总时长

这样做的好处是提醒更贴近“今天真实在用电脑”的体感，也尽量避免做得太重或太侵入。
